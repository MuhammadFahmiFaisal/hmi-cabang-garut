**[Web Resmi HMI Cabang Garut](https://github.com/muhammadfahmifaisal014/hmi-cabang-garut)**  
Official digital platform for HMI Cabang Garut serving as an information center and administrative tool.  
- **Stack:** Next.js 16 (App Router), React 19, Tailwind CSS v4, Supabase (Auth & DB).  
- **Features:** Interactive Public Portal, Comprehensive Admin Dashboard, News & Event Management, Member Database.  
- **Demo:** [Live App](https://hmicabanggarut.or.id/)

- **Stack:** React, TypeScript, Tailwind CSS, Recharts.
